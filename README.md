# Adani Health - Parivartaan Digital Nerve Centre

## Project info

This is the Parivartaan Digital Nerve Centre application for Adani Health.

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd frontend

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
